# 基础
```
    a = ['a', 'b', 'c']
    a.pop(0)=='a' # ['b', 'c']  如果index太大，会报错
    a.remove('b')==None  # ['c'], 如果不存在，会报错。既然你都知道remove什么了，就不返回给你remove了什么了
    
```


# 操作
* 切片
    ```
    a = [1,2,3]
    a[start:end]
    ```
    * 如果 `start >= len(a)`, 返回空列表

* zip合并
    ```
    zip([1, 2, 3], ['a', 'b', 'c'])
    (
        (1, 'a'),
        (2, 'b'),
        (3, 'c')
    )

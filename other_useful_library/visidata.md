
# visidata简介
[官网](https://www.visidata.org/docs/rows/)
vd 终端查看csv,excel文件, 非常方便

# [快捷键](https://jsvine.github.io/visidata-cheat-sheet/en/)
  * | 搜索列来选择行
  * s 选择行
  * " 筛选过滤出的结果
  * _ 适应宽度
  * Y zY 复制内容到系统剪贴板(需要xclip)

# 排序
`[`, `]`

# 数据类型
* `#` 把本列改成数字
* `~` 字符串

# 选择
* `s` 选择当前
* `gzs` 选择所有下面的, 按`"`, 筛选, z不知道含义

# 搜索
* `/` 搜索当前列
* `g/` 搜索所有列
* `c` 搜索列名

# 列操作
* `shift` + `C` 查看列名
通过编辑width来实现隐藏不需要的列

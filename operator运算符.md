# 运算符执行顺序
* 运算符	描述
* **	指数 (最高优先级)
* ~ + -	按位翻转, 一元加号和减号 (最后两个的方法名为 +@ 和 -@)
* * / % //	乘，除，取模和取整除
* + -	加法减法
* >> <<	右移，左移运算符
* &	位 'AND'
    * 注意，这个算然也可以对位的and进行操作，但是顺序却远大于 and
    * i==16 and j==19 和 i==16 & j==19 就是两种结果了
* ^ |	位运算符
* <= < > >=	比较运算符
* <> == !=	等于运算符
* = %= /= //= -= += *= **=	赋值运算符
* is is not	身份运算符
* in not in	成员运算符
* not or and	逻辑运算符
